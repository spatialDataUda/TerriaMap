Convenio Interinstitucional

Convenio de cooperación interinstitucional para la ampliación de base de datos, mantenimiento de sensores meteorológicos y generación automática de reportes para monitoreo de variables solares, eólicas, geológicas y climáticas en soporte a CELEC EP

Geoserver

Es un servidor de datos espaciales desarrollado para el manejar datos geoespaciales

Grafana

Herramienta libre que permite visualizar datos temporales de forma dinámica y flexible estableciendo métricas predeterminadas

TerriaJs

Plataforma web para la visualización y análisis de datos geoespaciales brindando al usuario la exploración, comparación y agregación de mapas de forma inmediata

Shiny

Es un paquete de R que permite construir aplicaciones web interactivas a partir de los scripts de R.

Objetivo

Integrar a la base de datos espacio-temporal la información de monitoreo de variables eólicas, realizar un diagnóstico de las necesidades de mantenimiento de los sensores meteorológicos, y fortalecer las capacidades de análisis de datos a través de la aplicación de análisis espacio-temporales, la creación de una plataforma web de visualización, y de funcionalidades de reportes periódicos automatizados para el monitoreo de variables solares, eólicas, geológicas y climáticas en soporte a CELEC EP en el Austro del Ecuador.
